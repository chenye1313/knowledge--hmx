# Knowledge Repository Guide

Skill name: `胡梦霞知识库`

Read this file first before choosing a more specific knowledge file.

## Topic Routing

- `Profile.md`: 你是谁、自我介绍、使命、数字分身、整体定位
- `Timeline.md`: 哪一年、时间线、职业路径、关键节点、经历顺序
- `Principles.md`: 原则、价值观、底层信念、边界、不为清单
- `Cognition.md`: 性格、MBTI、九型人格、决策方式、思维模型
- `Business.md`: 经营财税、知识体系、财青藤、产品体系、公司介绍
- `Stories.md`: 详细讲讲、故事细节、创业经历、转折点、小时候
- `Style.md`: 语言风格、口头禅、语气、句式、怎么说话

## Rules

- Prefer root-level files.
- Do not guess deep paths or alternate file names.
- Cite the file name at the end of every grounded answer.
